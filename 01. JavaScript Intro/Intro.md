# JavaScript (JS)

JavaScript, often abbreviated **JS**, is a versatile programming language that is one of the core technologies of the **World Wide Web**, alongside HTML and CSS. It enables developers to add **interactivity** to web pages, making websites dynamic and user-friendly.

## Features

- Add interactivity to web pages: sliders, alerts, click events, popups, and more.
- Works in both **browser** and **non-browser environments**.
- Supports **object-oriented**, **functional**, and **event-driven** programming styles.

## Environments Where JavaScript is Used

1. **Browser**
   - Runs directly in the browser.
   - Powers interactive UI features on websites.

2. **Node.js**
   - Allows writing **server-side code** in JavaScript.
   - Enables building APIs, web servers, and backend applications.

3. **Electron**
   - For building **cross-platform desktop applications** with JS, HTML, and CSS.

4. **React Native**
   - For building **mobile applications** for iOS and Android using JavaScript.

## Examples of JavaScript in Action

- Sliders, carousels, and popups on websites.
- Alerts, prompts, and confirmations in web apps.
- Server-side APIs with Node.js.
- Desktop apps like Visual Studio Code built with Electron.
- Mobile apps using React Native.

## Getting Started

To start using JavaScript in a web page:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>JS Example</title>
  </head>
  <body>
    <h1>Hello JavaScript!</h1>
    <script>
      alert("Welcome to JavaScript!");
      console.log("Hello from the browser console!");
    </script>
  </body>
</html>
```
